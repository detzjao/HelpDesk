# 🚀 HelpDesk Fullstack

Sistema de chamados com autenticação e dashboard.

## 🧠 Stack

* Node.js + Express
* SQLite
* React + Vite
* Tailwind CSS

## ▶️ Como rodar

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

## 🔐 Funcionalidades

* Login com JWT
* CRUD de chamados
* Filtro por status
