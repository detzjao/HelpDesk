# 🚀 HelpDesk Fullstack

Sistema de gerenciamento de chamados estilo SaaS.

## 🧠 Tecnologias

* Node.js + Express
* SQLite
* React (Vite)
* Tailwind CSS

## 🔐 Funcionalidades

* Login com JWT
* CRUD de chamados
* Filtro por status
* Dashboard moderno
* Contador de tickets

## ▶️ Como rodar

### Backend

cd backend
npm install
node src/database/init.js
npm run dev

### Frontend

cd frontend
npm install
npm run dev
